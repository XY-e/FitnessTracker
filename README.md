# FitnessTracker
## Overview
FitLife Tracker is a feature-rich fitness application designed to help users monitor and improve their health and wellness. It enables users to track workouts, manage diet logs, monitor progress through data visualizations, and set personalized fitness goals — all in one platform.

## Features
1. Authentication
- User Sign Up
- Secure Login
2. Homepage
- Clean and motivational landing page
- CTA for new users to join
3. Profile Management
- Editable user information (weight, height, goals, etc.)
- Saves personalized target weight & calorie goals
4.Exercise Log
- Add, edit, or delete exercise entries
- Search & filter tools
- Table for exercise history
- Pop-up “Add Exercise” window
5. Diet Tracking
- Log meal entries
- Nutrition breakdown: calories, protein, carbs, fats
- Daily summary
- View Log History & delete entries
6. Progress & Dashboard
- Weekly calories burned line chart
- Exercise targets with progress bars
- My Progress: pie chart, bar chart, line chart & summaries
7. Wearable Device Integration
- Sync devices such as Apple Watch, Fitbit, and Mi Band
- Real-time data visualization (steps, heart rate, sleep, hydration, calories burned)
8. Settings
- Update account information
